# Installing on Linux

1. Install python3 and download this repository.
2. Open folder.
3. Open terminal on folder or ```cd /home/user/type_directory_to_this_folder```
4. Type: ```chmod +x Chance.sh```
5. After 4. type this: ```./Chance.sh```
